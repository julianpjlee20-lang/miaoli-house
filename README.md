# 苗栗預售屋資訊網

一個用於測試現代網站架構的專案。

## 🏗️ 架構

```
┌─────────────────────────────────────────────┐
│                 VPS (Zeabur)                  │
│  ┌─────────────────────────────────────────┐│
│  │              Coolify                    ││
│  │  ┌─────────────┐  ┌────────────────┐ ││
│  │  │ Next.js     │  │  PocketBase    │ ││
│  │  │ (前端)       │  │  (後端 API)    │ ││
│  │  └─────────────┘  └────────────────┘ ││
│  │  ┌─────────────┐  ┌────────────────┐ ││
│  │  │ MinIO       │  │  Cron Job      │ ││
│  │  │ (文件儲存)   │  │  (資料更新)    │ ││
│  │  └─────────────┘  └────────────────┘ ││
│  └─────────────────────────────────────────┘│
└─────────────────────────────────────────────┘
```

## 📦 組成

| 組件 | 用途 |
|------|------|
| Next.js | 前端網站 |
| PocketBase | 用戶認證、數據儲存 |
| MinIO | 文件/圖片儲存 |
| Cron | 定時更新預售屋資料 |

## 🚀 部署

### 前置需求

- VPS (Zeabur 或其他)
- 網域

### 快速部署

```bash
# 1. 克隆專案
git clone <your-repo>
cd miaoli-house

# 2. 複製環境變數
cp .env.example .env

# 3. 啟動服務
docker-compose up -d
```

## 🔧 開發

```bash
# 前端開發
npm run dev

# 抓取資料
npm run fetch-data

# 部署
npm run deploy
```

## 📁 目錄結構

```
├── docker-compose.yml    # Docker 配置
├── frontend/            # Next.js 前端
├── backend/            # PocketBase 配置
├── scripts/           # 資料抓取腳本
├── minio/             # MinIO 配置
└── .env.example       # 環境變數範例
```
